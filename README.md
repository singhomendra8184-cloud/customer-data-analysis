
Problem Statement:
Analyze customer purchasing behavior to generate actionable business insights using data analytics and visualization.

Dataset Source:
Public retail dataset (Kaggle)

Tools and Technologies:
Python, SQL, Pandas, NumPy, Power BI, Advanced MS Excel

Steps Performed:
- Data Cleaning and Preprocessing
- SQL Queries for Data Extraction
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Interactive Dashboard Creation

Key Analysis Performed:
- Sales Trend Analysis
- Customer Segmentation
- Revenue Contribution Analysis

Results:
Improved KPI visibility and faster decision-making through automated dashboards.

Business Insights:
Top 20% of customers generated nearly 65% of total revenue.
Festive months showed a 35% spike in sales.

Future Improvements:
Add predictive sales forecasting and real-time data integration.
